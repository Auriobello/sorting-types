Quicksort

Algoritmo di ordinamento divide-et-impera efficiente

Introduzione

Quicksort è uno degli algoritmi di ordinamento più efficienti e ampiamente utilizzati. Utilizza il paradigma divide-et-impera, selezionando un elemento pivot e riorganizzando l'array attorno ad esso.

Complessità temporale: O(n log n) nel caso medio, O(n²) nel caso peggiore

Complessità spaziale: O(log n) dovuto alla ricorsione

Come Funziona

  1. Scegli un elemento pivot dall'array
  2. Riorganizza l'array in modo che tutti gli elementi minori del pivot siano a sinistra, e quelli maggiori a destra
  3. Ricorsivamente applica Quicksort alle due partizioni

Implementazione di Esempio

void quicksort(int arr[], int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quicksort(arr, low, pi - 1);
        quicksort(arr, pi + 1, high);
    }
}

int partition(int arr[], int low, int high) {
    int pivot = arr[high];
    int i = (low - 1);
    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(&arr[i], &arr[j]);
        }
    }
    swap(&arr[i + 1], &arr[high]);
    return (i + 1);
}

Questa è un'implementazione base di Quicksort in C. La funzione partition riorganizza l'array attorno al pivot.

Vantaggi e Svantaggi

Vantaggi:

  • Molto efficiente nella pratica
  • Ordinamento in loco (non richiede memoria aggiuntiva)
  • Cache-friendly

Svantaggi:

  • Caso peggiore O(n²) se il pivot è scelto male
  • Non stabile (l'ordine relativo di elementi uguali può cambiare)

Conclusione

Quicksort è spesso la scelta migliore per l'ordinamento generale grazie alla sua efficienza pratica. È implementato in molte librerie standard, incluso il C standard library (qsort).
