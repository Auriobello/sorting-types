Bogosort

L'algoritmo di ordinamento più inefficiente mai concepito

Introduzione

Bogosort è un algoritmo di ordinamento che funziona mescolando casualmente gli elementi dell'array fino a quando non risultano ordinati. È anche conosciuto come "stupid sort" o "slowsort" per la sua inefficienza.

Complessità temporale: O(n!)

Attenzione: Questo algoritmo non dovrebbe mai essere utilizzato in produzione!

Funzione ordinato

int ordinato(int v[], int n) {
    for (int i = 0; i < n - 1; i++) {
        if (v[i] > v[i + 1]) {
            return 0;
        }
    }
    return 1;
}

Questa funzione controlla se l'array è ordinato in ordine crescente. Scorre l'array e verifica se ogni elemento è minore o uguale al successivo. Restituisce 1 se ordinato, 0 altrimenti.

Funzione mescola

void mescola(int v[], int n) {
    for (int i = 0; i < n; i++) {
        int j = rand() % n;
        int tmp = v[i];
        v[i] = v[j];
        v[j] = tmp;
    }
}

Questa funzione mescola casualmente l'array. Per ogni posizione i, sceglie un indice j casuale e scambia v[i] con v[j]. Questo crea una permutazione casuale dell'array.

Funzione bogoSort

void bogoSort(int v[], int n) {
    while (!ordinato(v, n)) {
        mescola(v, n);
    }
}

Questa è la funzione principale dell'algoritmo. Continua a mescolare l'array fino a quando la funzione ordinato non restituisce vero (1). In teoria, prima o poi l'array sarà ordinato, ma in pratica potrebbe richiedere un tempo infinito.

Esecuzione del Programma

Il programma genera un array di 10 numeri casuali compresi tra 0 e 99, lo ordina utilizzando Bogosort e stampa i risultati.

Esempio di output:

Generazione array casuale per Bogosort:
45 23 67 12 89 34 56 78 90 1
Array non ordinato.

Array ordinato con Bogosort:
1 12 23 34 45 56 67 78 89 90
